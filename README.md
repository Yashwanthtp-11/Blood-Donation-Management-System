# Blood Donation Management System 
A comprehensive web-based platform designed to bridge the gap between blood donors and those in need. This system streamlines the process of blood registration, search, and inventory management for blood banks.
## Key Features
-   User Authentication: Secure login and registration for Donors, Recipients, and Admins.
-   Donor Management: Register donors, update profiles, and track donation history.
-   Real-time Search: Search for donors based on blood group and location.
-   Admin Dashboard: Manage donor records, blood stock levels, and user queries.
-   Blood Stock Tracking: Automated updates of available units in the blood bank.
-   Urgent Requests: Ability to post and view emergency blood requirements.
## Tech Stack
-   Frontend: HTML5, CSS3, JavaScript (Bootstrap for styling)
-   Backend: PHP 
-   Database: MySQL
-   Server: XAMPP / Apache

## Installation & Setup
Follow these steps to run the project locally:

1.   Clone the repository  
   git clone [https://github.com/Yashwanthtp-11/Blood-Donation-Management-System.git] (https://github.com/Yashwanthtp-11/Blood-Donation-Management-System.git)
2.	Setup Database
o	Open XAMPP Control Panel and start Apache and MySQL.
o	Go to http://localhost/phpmyadmin.
o	Create a new database (e.g., blood_bank).
o	Import the .sql file found in the project folder.
3.	Configure Connection
o	Open the database configuration file (usually db.php or config.php).
o	Update the database name, username, and password.
4.	Run Project
o	Move the project folder to C:\xampp\htdocs\.
o	Open your browser and go to http://localhost/BLOOD-DONATION-MANAGEMENT
